环境配置-macOS

 

在macOS设备上搭建Theos环境

首先介绍macOS平台，对系统当然是有要求的，macOS版本至少为10.9，并且需要安装Xcode 5.0以上（Xcode 4也行，但是这样你就只能写ARMv6的软件了 :-/），配置完成后，你便可

以使用theos给macOS，iOS，watchOS，tvOS开发软件。

 

开始准备

由于Xcode安装时默认没有安装Xcode Command Line Tool，所以我们需要安装一下，打开Terminal终端

输入xcode-select --install，弹框点击安装后，耐心等待。

接下来，你需要安装Homebrew（这个东西以后还会给你提供很多帮助）

在终端输入

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

macOS本身并没有dpkg，并且theos开发的软件需要用到ldid进行签名，所以我们需要安装dpkg和ldid，打开终端，输入brew install dpkg ldid，耐心等待。

 

安装Theos

我们习惯把theos安装在/opt/theos下，那么开始吧

sudo git clone --recursive https://github.com/theos/theos.git /opt/theos

这段命令需要你输入密码，密码看不到，输完按回车就好了

接下来修改一下权限

sudo chown -R $(id -u):$(id -g) /opt/theos

接下来，设置一下环境变量

export THEOS=/opt/theos（如果你安装到别的地方了就输入对应的路径）

 

完成安装￼

接下来，我们需要测试一下theos是否安装成功了

输入$THEOS/bin/nic.pl，如果你看到类似这样的东西，你的theos就已经安装完成了！

讲真大佬们已经帮我们解决了很多东西了，新版theos的安装较旧版来说，方便多了...
