环境配置-iOS

 

在iOS设备上搭建Theos环境￼

如果你想把你的iOS设备变成你随身携带的工作环境，那么请看这里（不推荐用主力机，因为很占空间…可以搞个4、500块的能越狱

的iPhone 5s，这个不错，至于空间，选个16G的对我这种闲鱼来说足够了:-)，相比买个Mac便宜了整整一位数）

然后，你需要越狱（废话，这也是我不推荐在日常使用的设备上安装的原因）

 

开始准备

Cydia中添加源

https://coolstar.org/publicrepo/

搜索安装Theos Dependencies和iOS Toolchain以及BigBoss Recommended Tools

别忘了还得在Cydia里选个终端软件（推荐White Terminal吧，有按钮看着好点）

你还需要个文件管理器，可以使用Filza File Manager

￼

安装Theos

我们选择安装到/var/theos/下 ，电脑用ssh链接到iOS设备（iOS 10设备自带了dropbear，如果你想直接连接到设备的话，可以在iOS

设备终端，登陆root后输入dropbear -R -F -p 22），或者直接在iOS设备上使用MTerminal，White Terminal之类的软件上输入

git clone --recursive https://github.com/theos/theos.git /var/theos

（如果你遇到这样的提示“Error: SSL certificate problem: unable to get local issuer certificate”，请先输入git config --global

http.sslVerify false ）

如果提示没有权限的话，请先登陆到root :-)

耐心等待下载完成。

 

安装SDK

接下来有两种方案

方案一、你需要下载一个Xcode，用解压软件打开，在Xcode里找到iOS SDK，放到设备的/var/theos/sdks/下

方案二、打开终端，输入

git clone https://github.com/theos/sdks /var/theos/sdks

 

 

设置环境变量、配置文件￼

打开终端，输入

touch ~/.profile

echo 'export THEOS=/var/theos' >>~/.profile

echo 'alias theos="$THEOS/bin/nic.pl"' >>~/.profile

source ~/.profile

ln -s /usr/local/bin/perl /usr/bin/perl

 

完成安装

接下来我们便可以来测试一下是否安装成功了，打开终端输入$THEOS/nic.pl如果你看到了这样的画面，👌，安装成功。
